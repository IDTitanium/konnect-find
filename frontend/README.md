# KonnectFind React Client

The React client provides an expressive desktop discovery experience and a
full-screen mobile swipe feed. It supports conversational and image search,
vendor discovery, product details, persistent cart state, stock-aware
quantities, checkout, order confirmation, and operator analytics.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

Vite proxies `/api` requests to `http://127.0.0.1:8000` by default. Override it
with `VITE_API_PROXY_TARGET`.

See the project-level [README](../README.md) and
[demonstration guide](../docs/DEMO.md).
